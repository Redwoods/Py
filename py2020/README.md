- **과제평가**
  - '#' : 과제 미제출 (-1)
  - '^' : 과제 코드 일부 오류 (-0.5)
  - '@' : 노력과 아이디어 인정!

- 과제 평가 기준
  - 미제출, 2문제 이상 코드 미흡: #, 1문제 코드 미흡: ^ , 모든 문제 코딩:@ , Best:@@
  
- 중간고사 과제 평가 기준
  - 2문제 이상 설명 미흡: , 1문제 설명 미흡:@ , 모든 문제 설명:@@ , Best:@@@
