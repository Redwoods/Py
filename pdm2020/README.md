# 생체신호 마이닝 (2021 2학기)

## Physiological Data Mining
* Physiological data
  - images & movies: X-ray, CT, MRI, PET, ...
  - signals: ECG, EEG, PPG, EMG, ...
  - text: 문진, DNA, SNS, ...
* Data Mining
  - 데이터 수집 (Acquisition)
  - 데이터 전처리(Preprocessing)
  - 데이터시각화 (Visualization)
  - 데이터 분석 (Analysis)
    * 특징 추출 및 머신러닝
    * 딥러닝
    * AutoML
  - AI 모델 기반 WEB 서비스
    * streamlit
